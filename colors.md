## Colors
### Color Names
```javascript
$('canvas').drawArc({
  fillStyle: 'darkblue',
  x: 50, y: 50,
  radius: 30
});
```

### Hex Values
```javascript
$('canvas').drawArc({
  fillStyle: '#009',
  x: 50, y: 50,
  radius: 30
});
```

### RGB Values
```javascript
$('canvas').drawArc({
  fillStyle: 'rgb(0, 0, 153)',
  x: 50, y: 50,
  radius: 30
});
```

### RGBA Values
```javascript
$('canvas').drawArc({
  fillStyle: 'rgba(0, 0, 153, 0.5)',
  x: 50, y: 50,
  radius: 30
});
```
